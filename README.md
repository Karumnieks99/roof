# Summit RoofCare — Demo Landing Page

A polished, responsive landing page for a fictional roofing company. Built as a portfolio piece to pitch roofing businesses in English-speaking markets (AU, UK, IE, CA, and similar).

**Live demo:** [karumnieks99.github.io/roof](https://karumnieks99.github.io/roof)

---

## What it is

A conversion-focused, mobile-first landing page built with clean HTML, CSS, and vanilla JavaScript — no frameworks, no build step, no dependencies.

Designed to show roofing companies what a professional web presence could look like for their business.

---

## Features

- Sticky header with mobile hamburger nav overlay
- Hero section with strong primary CTA
- Trust bar, services grid, before/after project cards
- Why Us, process steps, testimonials, service areas
- FAQ accordion
- Final CTA section
- Fully responsive (mobile → tablet → desktop)
- Smooth scroll, scroll-reveal animations, accessible markup

---

## Tech

| | |
|---|---|
| Markup | Semantic HTML5 |
| Styles | CSS custom properties, mobile-first media queries |
| Scripts | Vanilla JS (IntersectionObserver, no dependencies) |
| Fonts | [Oswald](https://fonts.google.com/specimen/Oswald) + [Lato](https://fonts.google.com/specimen/Lato) via Google Fonts |
| Deploy | GitHub Pages / Netlify / Vercel (no build step) |

---

## Design

- **Palette:** Charcoal `#1C1F26` · Amber `#E07B2A` · Warm off-white `#F5F3EF`
- **Approach:** Dark hero → alternating light sections → dark footer
- **Typography:** Oswald 700 for headlines, Lato 400/600 for body

---

## Usage

Clone or download, then open `index.html` in a browser. No installation required.

```bash
git clone https://github.com/Karumnieks99/roof.git
cd roof
# open index.html in your browser
```
